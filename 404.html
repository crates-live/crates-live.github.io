About

crates.live is a free website to browser Rust Cargo crates depdencies.

Features

  • Dependency Resolution crates.live engine does a full dependency resolution by matching the depedency version.
  • Interactive Chart
  • Image Export
  • Open API
  • Daily Updates

Tech Stack

crates.live uses a bunch of tech frameworks.

  • Rust crates.live backend and crawler is developed with Rust and open source Rust libraries.
  • Postgres the database of choice.
  • GraphQl Instead of REST, we use a GraphQl server powered by Rust.
  • VueJS Front-end Library.
  • Docker All executables are containerized with Docker.
  • Terraform Helps launch and maintain our infrastructure.
  • Github host repositories, build production and deployment actions.
  • Digital Ocean This website uses a xxx droplet.
  • Let's Encrypt To provide an SSL certificate.
  • Namecheap Domain name registration.

Rust Crates data

crates.live collects data from crates.io index and also by crawling crates.io API.

All crates on Crates.io are public, including the list of crate owners’ user names and the crate upload date. Anyone may view or download a crate’s contents. Because of the public nature of Crates.io, any personal data you might include in a Cargo.toml file uploaded to a crate will be publicly available.

For example, if an email address is in the `authors` field in the Cargo.toml file, that email address will also be public.

For more information, check the crates.io privacy policy.
